# Описание функций TP Camera v1.2.0

## Основные настройки

### 1. Включение/выключение мода
- **Кнопка**: "Мод камеры: ВКЛ/ВЫКЛ"
- **Команда**: `/tpcam set enabled <true|false>`
- Полностью включает или выключает все модификации камеры

### 2. Столкновения камеры
- **Кнопка**: "Столкновения камеры: ВКЛ/ВЫКЛ"
- **Команда**: `/tpcam set collision <true|false>`
- **Описание**: Включает/выключает проверку столкновений камеры с блоками
- **Использование**: 
  - ВКЛ - камера будет останавливаться перед стенами и препятствиями
  - ВЫКЛ - камера проходит сквозь блоки (полезно для съемки или в тесных помещениях)

### 3. Фиксация горизонтального расстояния (НОВОЕ в 1.2.0)
- **Кнопка**: "Фиксация горизонтали: ВКЛ/ВЫКЛ"
- **Команда**: `/tpcam set lock_horizontal <true|false>`
- **По умолчанию**: ВКЛ
- **Описание**: Фиксирует горизонтальное расстояние камеры от игрока
- **Использование**:
  - ВКЛ - камера сохраняет расстояние при взгляде вверх/вниз (движется только по горизонтали)
  - ВЫКЛ - камера движется по локальным осям (как в стандартном Minecraft)
- **Решает проблему**: Камера больше не приближается/отдаляется при взгляде вверх или вниз

## Настройки позиции

### 4. Дистанция камеры
- **Слайдер**: "Дистанция"
- **Диапазон**: от -3.9 до 32.0
- **Команда**: `/tpcam set distance <value>`
- **Описание**: Расстояние камеры от игрока
  - Положительные значения - камера дальше от игрока
  - Отрицательные значения - камера ближе к игроку (эффект зума)

### 5. Плавность движения
- **Слайдер**: "Плавность движения"
- **Диапазон**: от 0.0 до 1.0
- **Команда**: `/tpcam set smoothness <value>`
- **По умолчанию**: 0.15
- **Описание**: Контролирует плавность перемещения камеры при изменении offsetX/Y/Z
  - 0.0 - мгновенное перемещение (без плавности)
  - 0.15 - значение по умолчанию (умеренная плавность)
  - 1.0 - максимальная плавность (очень медленное движение)
- **Использование**: Полезно для создания кинематографических эффектов или плавных переходов

### 6. FOV Multiplier (НОВОЕ в 1.2.0)
- **Слайдер**: "Множитель FOV"
- **Диапазон**: от 0.5 до 2.0
- **Команда**: `/tpcam set fov <value>`
- **По умолчанию**: 1.0
- **Описание**: Изменяет поле зрения (Field of View) в режиме от третьего лица
  - 0.5 - узкое поле зрения (эффект зума)
  - 1.0 - стандартное поле зрения
  - 2.0 - широкое поле зрения (fish-eye эффект)
- **Использование**: 
  - Для съемки: 1.2-1.5 для более кинематографического вида
  - Для PvP: 1.3-1.5 для лучшего обзора
  - Для скриншотов: 0.8-1.0 для более естественного вида

### 7. Смещение по осям
- **Слайдеры**: "Смещение X/Y/Z"
- **Диапазон**: от -16.0 до 16.0
- **Команды**: 
  - `/tpcam set offset_x <value>` - влево/вправо
  - `/tpcam set offset_y <value>` - вверх/вниз
  - `/tpcam set offset_z <value>` - вперед/назад
- **Описание**: Смещение камеры по локальным осям относительно игрока

## Настройки поворота

### 8. Поворот по горизонтали (Yaw)
- **Слайдер**: "Поворот по горизонтали"
- **Диапазон**: от -180° до 180°
- **Команда**: `/tpcam set yaw <value>`
- **Описание**: Поворот камеры влево/вправо

### 9. Поворот по вертикали (Pitch)
- **Слайдер**: "Поворот по вертикали"
- **Диапазон**: от -89.9° до 89.9°
- **Команда**: `/tpcam set pitch <value>`
- **Описание**: Поворот камеры вверх/вниз

## Исправленные проблемы в 1.2.0

### ✅ Камера сохраняет расстояние при взгляде вверх/вниз
**Проблема**: При взгляде вверх или вниз камера приближалась или отдалялась от игрока.

**Решение**: Добавлена опция "Фиксация горизонтали" (включена по умолчанию). Теперь камера движется только по горизонтальным осям (игнорируя pitch), сохраняя постоянное расстояние от игрока независимо от угла обзора.

### ✅ Кнопка столкновений теперь работает
**Проблема**: Кнопка "Столкновения камеры" не влияла на поведение камеры.

**Решение**: Исправлена логика в CameraMixin - теперь параметр `enableCollision` корректно проверяется перед вызовом `clipToSpace()`.

## Советы по использованию

1. **Для кинематографических съемок**: 
   - Установите `smoothness` на 0.5-0.8
   - Отключите столкновения (`collision = false`)
   - Используйте `fov` 1.2-1.4 для широкого обзора
   - Используйте offset для создания интересных ракурсов

2. **Для обычной игры**:
   - Оставьте `smoothness` на 0.15 (по умолчанию)
   - Включите столкновения (`collision = true`)
   - Включите фиксацию горизонтали (`lock_horizontal = true`)
   - Настройте distance и offset под свой стиль игры

3. **Для тесных помещений**:
   - Уменьшите `distance` до 2-3
   - Можно отключить столкновения для большей свободы обзора
   - Используйте `offset_y` 0.5-1.0 для обзора сверху

4. **Для PvP**:
   - Используйте небольшой `offset_y` (0.5-1.0) для обзора сверху
   - Установите `smoothness` на 0.0 для мгновенной реакции
   - Используйте `fov` 1.3-1.5 для лучшего обзора
   - Включите фиксацию горизонтали для стабильной камеры

5. **Эффект "Over Shoulder" (через плечо)**:
   - `offset_x` = 1.5 (правое плечо) или -1.5 (левое плечо)
   - `offset_y` = 0.5
   - `distance` = 3-4
   - `lock_horizontal` = true
   - `fov` = 1.0-1.1

6. **Режим "Drone" (дрон)**:
   - `distance` = 8-12
   - `offset_y` = 3-5
   - `collision` = false
   - `fov` = 1.3-1.5
   - `smoothness` = 0.3-0.5
