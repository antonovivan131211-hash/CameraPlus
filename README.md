# Camera Plus (Fabric 1.20.1)
Мод добавляет гибкую настройку камеры от третьего лица с расширенными возможностями.

## Возможности
- Включение/выключение модификации камеры
- Настройки автоматически работают и в переднем виде от третьего лица
- Настройка параметров ползунками в реальном времени
- Смещение дистанции камеры (дальше/ближе)
- **Фиксация горизонтального расстояния** - камера сохраняет расстояние при взгляде вверх/вниз
- Смещение по локальным осям камеры: X / Y / Z
- Смещение поворота камеры (Yaw/Pitch)
- **Плавность движения камеры** - настройка плавности перемещения камеры
- **FOV Multiplier** - изменение поля зрения в режиме от третьего лица
- **Отключение столкновений** - возможность отключить проверку столкновений камеры с блоками
- Кастомный F5-прицел с корректным отображением точки прицеливания
- Отдельная функция динамики движения камеры (с включением и интенсивностью)
- Плавное приближение камеры при упоре в стену
- Сохранение параметров в `config/tpcamera.json`

## Управление
- Открыть GUI: клавиша `O`
- Сменить пресет динамики: клавиша `P`
- Открыть GUI через команду: `/tpcam gui`
- Все изменения в GUI применяются сразу без кнопки Save
- Показать текущие значения: `/tpcam show`
- Включить/выключить: `/tpcam toggle`
- Сбросить к дефолту: `/tpcam reset`

## Команды настройки
- `/tpcam set enabled <true|false>`
- `/tpcam set front_view <true|false>` (всегда будет использоваться режим `true`)
- `/tpcam preset <classic|combat|cinematic|next>` - быстрые пресеты динамики
- `/tpcam set collision <true|false>` - включить/выключить столкновения камеры
- `/tpcam set lock_horizontal <true|false>` - фиксация горизонтального расстояния
- `/tpcam set crosshair <true|false>` - включить/выключить F5-прицел
- `/tpcam set dynamic_motion <true|false>` - включить/выключить динамику движения
- `/tpcam set distance <value>`
- `/tpcam set smoothness <value>` - плавность движения (0.0 - 1.0)
- `/tpcam set fov <value>` - множитель FOV (0.5 - 2.0)
- `/tpcam set dynamic_motion_intensity <value>` - интенсивность динамики движения (0.0 - 1.0)
- `/tpcam set offset_x <value>`
- `/tpcam set offset_y <value>`
- `/tpcam set offset_z <value>`
- `/tpcam set yaw <value>`
- `/tpcam set pitch <value>`
