# Камера Плюс (Ткань 1.20.1)
Мод добавляет гибкую настройку камеры от третьего лица с расширенными возможностями.

## Возможности
- Включение/выключение модификации камеры
- Настройки автоматически работают и в переднем виде от третьего лица
- Настройка параметров ползунками в реальном времени
- Смещение дистанции камеры (дальше/ближе)
- **Фиксация горизонтального расстояния** - камера сохраняет расстояние при взгляде вверх/вниз
- Смѵщение по локальным осям камѵры: X / Y / Z
- Смещенность движения камеров (Рыскание/Тангаж)
- **Плавность движения камеры** - настройка плавности перемещения камеры
- **Множитель FOV** - изменение поля зрения в режиме от третьего лица
- **Отключение столкновений** - возможность отключить проверку столкновений камеры с блоками
- Кастомный F5-прицел с корректным отображением точки прицеливания
- Отдельная функция динамики движения камеры (с включением и интенсивностью)
- Плавное приближение камеры при упоре в стену
- Сохранение параметров в `config/tpcamera.json`

## Управление
- Графический интерфейс пользователя: клавиш `O`
- Сменить пресет динамики: клавиша `P`
- Открыть GUI через коменду: `/tpcam графический интерфейс`
- Все изменеия в GUI пЀименяются сразу без кнопкѸ Сохранить
- Показать текущие значения: `/шоу tpcam`
- Включить/выключить: `/переключатель tpcam`
- Сбросить к дефолту: `/сброс tpcam`

## Команды настройки
- `/tpcam установлен включенным <истина|ложь>`
- `/tpcam установить вид спереди_ <истина|ложь>` (всегда будет использоваться режим `истинный`)
- `/предустановка tpcam <классический|бой|кинематографический|следующий>` - быстрые пресеты динамики
- `/столкновение набора tpcam <истинный|ложь>` - включить/выключить столкновения камеры
- `/tpcam установить lock_horizontal <истина|ложь>` - фиксация горизонтального расстояния
- `/перекрестие набора tpcam <истинный|ложь>` - включить/выключить F5-прицел
- `/tpcam установить динамическое_движение <истина|ложь>` - включить/выключить динамику движения
- `/tpcam установить расстояние <значение>`
- `/tpcam установить гладкость <ценить>` - плавность движения (0.0 - 1.0)
- `/tpcam установить поле зрения <значение>` - новое поле зрения (0,5 - 2,0)
- `/tpcam установить dynamic_motion_intensity <значение>` - интенсивность динамики движения (0.0 - 1.0)
- `/tpcam установить offset_x <значение>`
- `/tpcam установить offset_y <значение>`
- `/tpcam установить offset_z <значение>`
- `/tpcam установил рыскание <значение>`
- `/tpcam установить шаг <ценить>`
