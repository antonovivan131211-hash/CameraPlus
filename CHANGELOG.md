# Changelog
## [1.3.0] - 2026-03-01

### Добавлено
- Кастомный F5-прицел с переключателем в меню.
- Пресеты динамики: `Classic`, `Combat`, `Cinematic`.
- Быстрое переключение пресетов: клавиша `P` и команда `/tpcam preset <classic|combat|cinematic|next>`.
- Новый блок пресетов в GUI с отображением текущего режима (`Custom` при ручной настройке).
- Обновленный визуальный стиль меню: секции, улучшенный фон, более читаемая структура параметров.

### Исправлено
- Исправлена логика прицеливания в F5: прицел теперь корректнее совпадает с направлением взгляда аватара.
- Исправлены проблемы с применением столкновений камеры с блоками.
- Исправлены анимации/динамика камеры при движении и поворотах.
- Устранены рассинхроны управления и направления обзора в режиме камеры от третьего лица.
- Исправлен краш при загрузке, связанный с `CameraMixin$DynamicMotion` (некорректная ссылка на mixin-класс).

### Изменено
- Брендинг интерфейса обновлен на `Camera Plus` (название в UI и локализациях).
- Обновлено описание мода для публикации:
  `Гибко настраиваемая камера для Minecraft. Меняй FOV, смещение, плавность, поведение в 1/3 лице и создавай идеальный вид для игры, строительства и съёмки.`
